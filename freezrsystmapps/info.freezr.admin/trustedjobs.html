Trusted Jobs

Approve a job declared by one of your installed apps to run in-process on this server ("trusted job"). Trusting copies the job's code into the server's trusted-jobs area and records which users may run it. Only trust code you understand — a trusted job runs with server privileges.

This is like adding a node module to the server. Only add if you totally trust it. If you dont know what a node module is, get out of this page immediately!
