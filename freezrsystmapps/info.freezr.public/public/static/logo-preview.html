freezr logo + spinner + corner-button preview

Production class is .freezr-logo-btn-negative-gradient (B4) — defined in freezr_core.css and used by #freezer_img_button (the corner button that freezrApiV2.js drops onto every app page). Other variants on this page are preview-only (prefixed .preview-btn-*) so the global stylesheet stays lean. To swap the production button to a different variant later: copy its class block from this page's <style> into freezr_core.css and update the className in freezrApiV2.js _createElements().
